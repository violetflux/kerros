# Erste Schritte

## Installation

```bash
bun add kerros
```

Kerros unterstützt React 17, 18 und 19.

## Store erstellen

`createStore` erhält einen React Hook und gibt den Consumer Hook sowie den passenden Provider zurück.

```tsx
import { createStore } from 'kerros'
import { useState } from 'react'

export const [useCounter, CounterProvider] = createStore(() => {
  const [count, setCount] = useState(0)
  return { count, setCount }
})
```

Provider einbinden und benötigte Felder auswählen:

```tsx
function Counter() {
  const { count, setCount } = useCounter(s => ({
    count: s.count,
    setCount: s.setCount,
  }))

  return <button onClick={() => setCount(count + 1)}>{count}</button>
}
```

Selektoren dürfen inline stehen und benötigen kein `useCallback`.
