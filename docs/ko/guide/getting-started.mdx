Kerros는 표준 npm 패키지이며 npm, pnpm, Yarn, Bun에서 모두 사용할 수 있습니다. Kerros 저장소가 사용하는 패키지 매니저는 애플리케이션을 제한하지 않습니다. React 17, 18, 19를 지원합니다.

## Store 만들기

`createStore`는 React Hook을 받아 소비자 Hook과 대응하는 Provider를 반환합니다.

```tsx
import { createStore } from '@violetflux/kerros'
import { useState } from 'react'

export const [useCounter, CounterProvider] = createStore(() => {
  const [count, setCount] = useState(0)
  return { count, setCount }
})
```

Provider를 마운트하고 필요한 필드를 선택합니다.

```tsx
function Counter() {
  const { count, setCount } = useCounter(s => ({
    count: s.count,
    setCount: s.setCount,
  }))

  return <button onClick={() => setCount(count + 1)}>{count}</button>
}
```

selector는 인라인으로 작성할 수 있으며 `useCallback`이 필요하지 않습니다.
